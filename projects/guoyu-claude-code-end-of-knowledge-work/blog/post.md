## 一、为什么是从 2025 年 10 月开始，而不是 ChatGPT

郭宇把整个 AI 革命的起点重新划了一刀：

- **不是** 2022 年 12 月的 ChatGPT。ChatGPT 让 AI 成为对话工具、生活的一部分，但用户增长在两年后已经停滞，市场开始怀疑是不是泡沫。
- **而是** 2025 年 10 月，Anthropic 给 Claude Code 加上 **Skills（技能）** 这个机制的那一刻。

**Skill 是什么？** 一句话：**把你的工作流程写成一个 Markdown 文档**——早上起来干什么、读完邮件干什么、开完会决定做什么——然后 Claude Code 会自己根据这个文档现写代码、调用工具、把这件事做完。

这一步跨过了一个关键门槛：

> 程序员可以**不再审查**模型写出来的代码，它就能正常工作。

到这里，软件工程的"想清楚需求 → 产品经理 → 技术文档 → 写代码 → 发布"的流水线，被压扁成了"说一句话 → 它自己写"。

## 二、"软件的终结"：写死的代码 vs 流动的 Token

郭宇把软件分成两类：

| 类型 | 形态 | 例子 |
| --- | --- | --- |
| **写死的软件** | 工作流提前设计、代码固定、所有用户用同一个 | WordPress、Salesforce、Adobe、各种 SaaS |
| **最终的软件** | 没有固定代码，运行时根据需求**现写**，每次都是为你这一次任务定制的 | Claude Code、Claude Cowork、OpenClaw |

你以后看到的"软件"很多都不再是固定代码，而是**动态流动的 Token**——这些 Token 需要 GPU 推理，所以英伟达造卡 → 数据中心插电 → 模型现写代码 → 完成你这一次的需求。

**结果**：高度个性化 + 高度自动化。

> "我处理图片不再需要 Adobe，处理文档不再需要 SaaS。我直接告诉 Claude Code 我的需求，它用我电脑里的文件、我自己登录的账号、存到我自己的云盘里。我为什么还要用别人写的软件？"

这就是为什么近一年只要 Anthropic 发个新功能，美股就总有几家 SaaS 公司单日跌 20%。Claude 的 Legal 插件 2 月上线那天，相关法律科技股加起来蒸发了 ~2850 亿美元。

## 三、现场 Demo：让 Claude Cowork 重做一个个人网站

为了证明这不是 PPT，郭宇当场拿对谈嘉宾的个人网站做演示：

1. 打开 Claude Cowork 桌面端，让它读取硬盘里相关文件。
2. 用 Typeless 之类的语音输入直接说话："帮我做一个介绍 XX 记者写作的网站。"
3. 它自动调用 Web Search 抓到现有的 changche.com，理解作者背景（非虚构作家、牛津哲学硕士），列出代表作。
4. 让它再加载一个名为 `ui-ux-pro-max` 的 **Skill**——这是别人写好的、封装了顶级设计师方法论的"技能包"。本地没装？它**自己上网搜、自己装**。
5. 它自己一次装了 7 个设计相关 Skills，先建立一套设计系统，再用 Python 现写代码，**两三分钟出一个完整网站**，自动写出"From Shanghai to Seoul to Tokyo"这种你从没说过、但它从公开资料里读出来的细节。
6. 想做中日英三语版、想加 3D 动画、想模仿某个现成网站的设计语言？继续说话就行。

> "搜索引擎的用户已经不再是人，而是它。"

## 四、一个一个行业拆：谁先没了

### 1. 程序员，尤其是初级程序员
矽谷很多公司**已经完全停止招聘初级工程师**——招进来不如直接让 Claude Code 写，更快、更好、最重要的是**不用和人沟通**，省掉了组织里最大的摩擦。

短期：你一个人抵一万个人。
长期：根本不需要你。

### 2. SaaS（Salesforce、HR 系统、各种企业软件）
所有"功能可以被现写代码替代"的 SaaS 都在被定价。Adobe、各类设计工具、各类企业软件、博客平台——**WordPress 这种，从原理上未来会倒闭**，虽然像 jQuery 一样大概率还有很多人在用。

### 3. 法律
2026 年 2 月 Claude 发布 Legal Plugin 当天，彭博估算引发 **~2850 亿美元** 法律科技板块抛售。"企业内部法务团队**直接整建制裁掉**用 Agent 替代"已经在发生。

更激进的视角来自 2003 年的纪录片《Zeitgeist》：**如果世界上所有车都是自动驾驶，就不再需要交通法规和保险公司**。郭宇本科学的是政治学——政治和法律本质是为了化解"人和人、人和群体、群体和群体的摩擦"。当 Agent 之间能在共享记忆里直接对齐，公司 A 和公司 B 不需要开会、不需要对账，**法律可能依然存在，但已经没有了执行的意义**。

### 4. 知识分子 / 出版 / 写作
"所有知识分子都没有了。" 你可以让 Claude Code 扮演任何小说家、任何哲学史博士、任何美学专家。郭宇自己写书也用 Claude 做研究——"以前 Google 一搜只有屏幕这一页，现在 Claude Code 一次把整个互联网都翻一遍。"

### 5. 医疗 / 律师事务所等
助理岗位先没。复杂手术、出庭辩护这种"人和人的接触"还会保留一段时间。

### 6. 最后扩到所有行业
随着机器人发展，蓝领也会被波及。**只是知识工作者最先**。

## 五、中美之间的"6 个月时差"

郭宇直接给了一个具体数字：

> "中美在编码模型上有 **6 到 12 个月** 的代差。看起来很短，但在今天 6 个月就是天差地别。"

为什么？

- **美国缺电、中国缺卡**。
- 但中国公司（字节、阿里、腾讯、百度）买到的卡，**主要拿去做消费级多模态推理**——豆包、即梦、视频生成、推荐——因为他们要做"7 岁到 70 岁都能用、能赚钱"的产品。
- 真正能"现写代码"的编码模型训练，**没有分到足够的卡**。
- Kimi、MiniMax 这些公司在追，但路径主要是**蒸馏 Claude Opus**——蒸馏天然意味着永远晚一代。
- 字节是国内做得最好的之一，依然落后约半年。

英伟达 Rubin 架构上线之后，明年后年推理成本会降到现在的 1/10，"几乎免费的推理"会被解锁——这才是真正的变化普及到所有人时刻。

## 六、One-Man AI：一个人可以同时开 1000 家公司

郭宇正在做一个叫 **one man AI** 的产品（致敬日本的"一人电车")。

每天晚上他在自己的 5090 PC 上让它跑：

> "昨天跑了 1000 个循环，6 个多小时。我让它设计一家在东京做装置艺术的公司——完整财务计划、品牌网站、3D 装置作品（用 3GS）、然后自动起草邮件，发给东京所有相关画廊和需要装置艺术的公共空间。"

如果他一个人都能做到，那就意味着：

- 任何一个人，只要提出**一个想法**，Agent 可以同时把这个想法的 1000 种可能性全部实现并去执行。
- 失败的，按预设清算退出；不失败的，自动跑下去直到盈利。
- 创业不再需要 800 万日元注册资金、不再需要请人、不再需要"专心一个想法"。

软件的终结，意味着**时间也被压扁了**——所有可能性可以同时发生。你只需要选一个分支，甚至不选，让它全自动跑。

## 七、生产力起飞 vs 生产关系卡死：一个不稳定的未来

郭宇没有装乐观。他直接指向美股最近一段时间的撕裂：

- **生产力**确实大爆发：所有用 AI 的公司利润和收入都在暴涨。
- 同时也在**疯狂裁员**。
- 工作岗位减少 → 消费者减少 → 经济失去活力 → **走向通缩**。
- AI 让东西更便宜、服务更便宜，**进一步加剧通缩**。
- 生产关系（工资、雇佣、分配）还停留在工业时代，没有改写。

> "一方面会很糟糕，很多人失业、没有收入。
> 另一方面，他还可以订 200 美元 / 月的 Claude Code，做很多有意思的事情，也许能赚到钱，也许不行。
> 所以现在市场在定价的，就是这么一个非常不稳定的未来。"

主持人问："那大家干嘛呢？"
郭宇答："**可以去海边冲浪。**"

## 八、给年轻人的建议

> "如果你弟弟还没开始学写代码，**就别学了**。"

学什么？

- **接下来 3–6 个月**：学会用 Claude Code / Cowork 这一类编码 Agent。这是当前最高 ROI 的杠杆。
- **再往后**：连这个都不需要学。因为整个流程会进一步**全自动化**——发散想法本身（Auto Research）也会被 Agent 接管。

教小孩用 ChatGPT 的家长另外注意一件事：很多孩子已经不和父母聊心事，全都跟 ChatGPT 说。OpenAI 的模型在性格上**会顺着你**，你开心它就跟你开心——**对成长中的孩子不一定是好事**。

## 九、OpenClaw（"龙蝦")：常驻的本地 Agent Manager

最后一段是关于国内最近很火的 **OpenClaw**。

- 它**不是大语言模型**，而是一个 **Agent Manager / 编排器**——常驻在你自己的电脑（Mac mini 或 PC）里。
- 你给它接入语言模型（最强的就是 Claude Opus 4.6），它在背后自动调度多个 Agent。
- 自带 Heartbeat（心跳），24×7×365 一直活着。
- 通过微信、Telegram 等聊天软件就能跟它说话，它接管你的电脑全自动做事。
- 这种本地 Agent 一旦跑通，**必然**会上云——云端伺服器更强、推理更快、不断电、内存更多。
- **风险提示**：本地版已经有人被其他 Agent 骗钱的真实案例。**云端 Sandbox 模式更安全**，跑完即扔，不会改坏文件。
