# EP20 · Microsoft 大佬：AI 永远不会取代程序员

C# 和 TypeScript 之父、Microsoft technical fellow Anders Hejlsberg 在 Sajjaad Khader 的频道上聊编程和 AI 的未来。13 分钟，干货密度高。

主要观点：

- **AI 不会取代程序员**：AI 本身不会创新，CPU、操作系统、编程语言、AI 调用的 framework，这些都得人来设计。
- **Vibe coding 没那么神**：to-do list 这种被训练集见过几十亿次的东西 AI 能玩，但一旦要写新业务逻辑、发明新东西，就得回到人类。
- **学编程仍然值得**：编程语言是 AI 表达自己的方式，AI 默认底下那个世界已经存在。
- **新人最常见的错误**：投入时间不够。这个行业不是这样进步的。
- **C# / TypeScript 幕后故事**：从他几十年经验聊起。
