# EP36 · Kubernetes 与功成身退：Kelsey Hightower 的非典型工程师之路

> The Pragmatic Engineer 2026-06-03 期，主持 Gergely Orosz × 嘉宾 Kelsey Hightower（前 Google Distinguished Engineer、Kubernetes 早期布道者，已"光荣退休"两年）。

约 3 小时 23 分的长访谈。Kelsey 是个传奇——19 岁辍学去装 DSL 调制解调器，从麦当劳一路做到 Google 杰出工程师；Kubernetes 早期的诸多 talk、教程都是他做的；然后 42 岁主动从 Google 退休，到处做顾问、不再每天写代码。

聊到的几条主线：
- 自学路径：从装宽带、AutoCAD 到拿起 Ruby/Python 之前先吃透 shell 和 Unix
- Kubernetes 走到今天的设计哲学：「不是给开发者用的，是给写平台的人用的」
- 为什么选择在职业巅峰离开 Google：当工作不再让你成为更好的人，就该走
- 关于 AI 与软件工程：「我一直努力让自己看法更积极一点」——agent、文档、抽象层级的提升与失落
- 顾问关系怎么做才健康（包括 0.25% 股权这样的细节）
